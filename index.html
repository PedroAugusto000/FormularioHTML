PERGUNTAS E RESPOSTAS

  1. Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?

    2. Na tag input com o atributo type="text' o atributo "placeholder" serve para exibir um texto de ajuda, ele aparece em cima de onde se está escrevendo.


  2. Na tag <input> com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

    3. O atributo checker serve para serve para pré-selecionar um botão do atributo type="radio" quando a página é carregada. A caixa de seleção será marcada como padrão.


  3. Na tag <input> para que serve o atributo "value"?

    4. Serve para definir um valor padrão para a entrada ou um valor inicial do controle.


  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

    5. O atributo que deve ser utilizado é o "name".


  5. Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?

    6. Serve para associar um rótulo a um campo de entrada. O atributo for faz uma espécie de ponte entre o "input" e o "label", o atributo for aponta para o ID do input que se deseja associar.


  6. Na tag <option> para que serve o atributo "selected"?

    7. Serve para pré-selecionar uma opção em uma lista suspensa, como na escolha de uma UF em um site formulário.


  7. Nas tags de formulário para que serve o atributo "required"?

    8. Serve para especificar que o preenchimento de um campo é obrigatório.


  8. Para que serve a tag <textarea>?

    9. Serve para criar uma área de texto que pode ter várias linhas.


  9. Na tag <form> para que serve o atributo "action"?

    10. É usado para especificar o URL do servidor que irá processar os dados do formulário quando ele for enviado.


  10. Na tag <form> para que serve o atributo "method"?

    11. Especifica o método HTML a ser usado ao enviar o formulário.


  11. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form>?

    12. O valor que deve ser atribuido é o "POST", ele envia os dados escondendo-os da URI.


  12. Qual valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos?

    13. Para envio de arquivos também se dever usar o valor "POST"


  13. O que as tags <fieldset> e <legend> produzem?

    14. Ambas as tags são usadas para agrupar vários campos de entrada
